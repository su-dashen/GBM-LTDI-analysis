# Pre-release checklist

- [ ] Confirm that the public code matches the final analysis version used for the manuscript.
- [ ] Replace all example paths in `run_example.m`.
- [ ] Verify the correct MNI template for the MRI modality used for normalization.
- [ ] Verify label values and label names against the segmentation protocol.
- [ ] Confirm that the streamline atlas license permits public reference and redistribution.
- [ ] Add the manuscript citation and DOI to `CITATION.cff`.
- [ ] Confirm the copyright holder and license with your institution.
- [ ] Confirm that no patient data, protected health information, credentials, or local paths are tracked by Git.
- [ ] Create a tagged GitHub release matching the manuscript version.
